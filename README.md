# Do Not Feed the Monkeys BepInEx Mods

A small collection of BepInEx mods for **Do Not Feed the Monkeys**.

## Downloads

Download the mod you want from the **Releases** page.  
Each mod is provided as a separate `.zip` file, so you can install only the ones you want.

## Available Mods

### FastForward

FastForward lets you speed up the whole game.

To use it, hover your mouse over the in-game clock and hold the left mouse button.  
While the button is held down, the game runs faster.

### SecondaryMonitorDialog

SecondaryMonitorDialog adds dialog text to the secondary monitor view.

You can also interact with the dialog text directly in the secondary monitor. This makes it possible to follow one dialog on the main monitor and another dialog on the secondary monitor at the same time, without constantly switching between camera views.

## Installation

1. Download the `.zip` file for the mod you want.
2. Extract the downloaded `.zip`.
3. Open the extracted folder.
4. Copy all included files and folders into the game folder, where the game `.exe` is located.
5. Start the game.
